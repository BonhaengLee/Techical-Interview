# 이벤트 위임에 대해 설명해주세요 #

이벤트 위임은 이벤트 핸들러를 하위 요소에 추가하는 대신 상위 요소에 추가합니다.  
DOM 이벤트 버블링으로 인해 하위 요소에서 이벤트가 발생될 때 실행됩니다.

## 장점 ##

- 각 하위 요소에 이벤트 핸들러를 연결하지 않고,  
`상위 요소`에 하나의 `단일 핸들러`만 연결하므로 메모리 사용이 줄어듭니다.
- `제거`된 요소에서 핸들러를 `해제`하고,  
`새` 요소에 이벤트를 `바인딩`하는 연산이 필요없습니다.
